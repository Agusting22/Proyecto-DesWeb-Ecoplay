**Ecoplay** es una tienda online de productos gamer y tecnología. Desarrollada como trabajo práctico para la materia Diseño Web.

## Páginas

- `index.html` — Inicio con carrusel, categorías y productos destacados
- `pages/productos.html` — Catálogo completo de productos
- `pages/contacto.html` — Formulario de contacto

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
